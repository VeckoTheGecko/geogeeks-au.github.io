1. Make a web page

  1. Open a text editor
  2. Add the following: 
    <!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width,initial-scale=1.0">
        <title>Example</title>
    </head>
    <body>
        <h1>Example</h1>
    </body>
</html>
  3. Save as index.html

2. Add a map to it

Download and unpack leaflet ttp://leafletjs.com/ current version: http://cdn.leafletjs.com/leaflet/v1.0.3/leaflet.zip Turn webserver on, pointing to working directory. Mac & Linux users have python already, need to install for Windows users. Start: python -m http.server --cgi 8000 View empty leaflet map in web browser.
