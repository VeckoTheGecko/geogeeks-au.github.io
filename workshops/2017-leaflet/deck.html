Make a web page

  1. Open a text editor
  2. Add the following: 
    <!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width,initial-scale=1.0">
    <title>Example</title>
  </head>
  <body>
    <h1>Example</h1>
  </body>
</html>
  3. Save as index.html
